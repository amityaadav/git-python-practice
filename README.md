# git-practice

## Fundamentals
Working/Local Directory
(files in your workstation/computer)   

git add: Select which changes to include in your next commit
(Staging Area)

git commit: Saves to .git/ (local repo on your computer)
(Local Repo)

git push: Sends to GitHub (remote)
(Send your changes to remote)

## Fetch vs. Pull
Fetch to download and then review before you merge to your local
Pull to download and merge to your local

Fetch = check your mailbox
Pull = check mailbox + open and read the mail
